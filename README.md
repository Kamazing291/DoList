# Instructions!: 

1. Place the jar in a folder of your choice
2. Create a new folder for all your to-do lists (can be same folder)
3. Right where the jar is present, create a text file named "loc.txt"
4. Edit loc.txt and put the lists folder location in it, Ex. D:/Lists/  The last slash is important!
5. Test if everything works, and you're good to go :)
